## Run locally
### Prerequisites
- nodejs v22

### 1. Install packages
```shell 
  npm install 
```

### 2. Build 
```shell
  npm run build
```

### 2. Start
```shell
  npm run start
```

## Build executables
For Windows:
```shell
  electron-builder --win
```
or
```shell
  npx electron-builder -w
```
